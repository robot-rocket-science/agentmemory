## Project: agentmemory

**Status:** Research phase, no production code.

### What's been validated
| Area | Key Result |
|------|-----------|
| Bayesian retrieval | Thompson + Jeffreys, ECE=0.053, passes |
| Correction detection | V2: 26% -> 92% on real overrides (zero-LLM) |
| HRR encoding | Works experimentally, but real value is fuzzy-start traversal, not multi-hop |
| Two-hop HRR (S^2) | Failed -- near-saturated at 85% capacity; correct path is iterative BFS |
| Granular decomposition | 86% token reduction with sentence-level nodes |
| Context compression | Hybrid: CLAUDE.md + status injection + compactPrompt |
| Scaling | Degrades at 10K beliefs; hierarchical propagation helps |
